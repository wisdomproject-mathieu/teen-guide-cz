# Project Memory

## Core
Monkey Mind — teen mental health app in Czech. Target: cool teens, not kids.
Streetwear monkey characters throughout UI. Dark theme with neon accents.
ElevenLabs TTS with browser fallback. Lovable Cloud backend.
Freemium: 20% free / 80% premium. Never gate SOS/safety. No ads.
Pricing: Free, 99 CZK/mo, 799 CZK/yr. 7-day trial. Never >999 CZK/yr.
