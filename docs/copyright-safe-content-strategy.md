# Monkey Mind Copyright-Safe Content Strategy

## Core rule

Use YouTube as:
- inspiration
- distribution
- optional embedded viewing

Do not use YouTube as:
- a source of ripped audio
- a source of copied transcripts stored in the app
- a source of cloned voices without permission

## Product principle

The strategy docs are right: static audio alone is not enough for teens. The answer is not copyright risk. The answer is a stronger content stack:

1. original Monkey Mind scripts
2. short-form motion/video formats
3. licensed or owned voice
4. curated external embeds for discovery and social proof

## Safe content lanes

### 1. Original Monkey content

This is the main product.

Allowed:
- write original Czech scripts inspired by themes from motivational content
- adapt by mood, reason, and intensity
- generate original audio with licensed TTS or owned voice
- create Monkey avatar shorts from original scripts

This is the safest and strongest premium asset because you own it.

### 2. Curated YouTube embeds

This is the safest way to use third-party video.

Allowed:
- link to the original video
- embed the official YouTube player
- add your own caption, mood tag, or intro around it
- recommend a video as "watch next" content

Not allowed:
- extracting the audio into native playback
- republishing transcript text as if it were your script
- presenting the creator's work as Monkey Mind content

Use case:
- free discovery layer
- premium companion layer around the embed

### 3. Licensed creator packs

Best long-term option.

Allowed with permission:
- a Czech creator records original speeches for Monkey Mind
- creator grants app usage rights
- creator voice is used directly or cloned only with written permission

This is the strongest future premium lane because it combines:
- authenticity
- distribution
- legal clarity

### 4. Public-domain / creator-permission educational modules

Use for "Pochop to" style content.

Allowed:
- original summaries of psychology ideas
- original educational scripts
- explicitly licensed source material

## What to do with YouTube right now

For each promising YouTube video:

1. extract themes manually
2. write new Monkey Mind scripts from scratch
3. tag each script by:
   - mood
   - reason
   - intensity
   - format
   - free vs premium
4. optionally add the original video as an official embed

This gives you:
- no copyright mess
- stronger personalization
- better premium differentiation

## "Audio is dead" interpretation

The strategy docs do not mean "remove all audio."

They mean:
- audio-only is not enough as the main experience
- teens respond better to motion, direct address, short-form pacing, and visual identity

So the right stack is:
- text for speed
- short audio for low-friction rescue
- short video for premium engagement and sharing

## Best-in-class experience for teens

### In-app

- under 90 seconds from opening to value
- intensity-aware recommendations
- original Monkey speeches
- premium AI chat
- premium video shorts

### Outside the app

- TikTok / Shorts clips
- PWA sharing
- embedded official creator videos
- branded warrior share cards

## Immediate build priority

1. Keep original speeches as the product core
2. Add a curated external video layer using official embeds only
3. Build short-form Monkey video content from original scripts
4. Reserve premium for:
   - AI chat
   - full library
   - video shorts
   - advanced insights

## Red lines

Never:
- download YouTube audio for native app playback without rights
- clone a creator voice without explicit permission
- bulk-copy transcripts into premium packs
- hide third-party source attribution when using embeds

## Decision rule

If the content should feel like it belongs to Monkey Mind, create it originally.

If the content should stay clearly attached to the original creator, embed it officially.
